Posted on 2017-03-22 | Comments:

1. 文件操作

file(‘文件名’,’操作符’)
省略操作符 默认为读

模式 操作符 说明
只读 r 文件必须存在
只写 w 文件不存在则创建，文件存在则清空文件内容
追加 a 文件不存在则创建，文件存在则在文件末尾添加内容
读写 r+
写读 w+
追加 a+

在python中，以什么样的方式出来文件是比较重要的，如果你
以只读的方式打开文件，那你就不能写操作
以只写的方式打开文件，那你就不能读操作
以追加的方式打开文件，也应该是不能进行读操作的
操作完毕，不要忘记关闭文件

1.1 以只读方式打开文件

1.2 以只写方式打开文件

如果文件存在并且有内容，则会把文件之前的内容清除；如果文件不存在。则会创建文件

1.3 以追加的方式打开文件

这个时候，打开文件，刚才写入的内容并没有写入到文件里。

文件在关闭的时候才会写入到文件里或者强制写入

追加模式也不可以读

拓展

  1. 去掉换行符
    line.strip('换行符')
    省略换行符默认为空格

  2. 把行分割
    line.split('分隔符')

方法 说明
f.readlines() 以列表的形式读出文件
f.read() 以字符串的形式，一次性读出文件
f.seek() 跳到文件的指定位置
f.tell() 当前位置
f.truncate(100) 从头开始截断文件
f.writelines() 多行写
